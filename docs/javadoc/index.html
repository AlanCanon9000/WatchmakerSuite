Watchmaker Suite Javadoc

The Java port of Blind Watchmaker is divided into several projects, each of interest to different developers.

Watchmaker (core classes)

This project defines the core classes of the Watchmaker Suite, not including any morph-specific or GUI platform specific classes.

WatchmakerDawkinsBiomorphs (Common classes used by Dawkins' biomorphs)

This project defines classes common to Dawkins' original biomorphs: Monochrome, Colour, Arthromorphs, and Snails.

Dawkins' Biomorphs: Morph-specific class libraries

  • WatchmakerDawkinsMono (Classes specific to Dawkins' Monochrome Biomorphs)
  • WatchmakerDawkinsColour (Classes specific to Dawkins' Colour Biomorphs)
  • WatchmakerDawkinsSnails (Classes specific to Dawkins' Snails or 'Conchomorphs')
  • WatchmakerDawkinsArthromorphs (Classes specific to Dawkins' Arthromorphs)
  • WatchmakerDawkinsMorphsSwing (Classes specific to the Swing implementation of Dawkins' Biomorphs)

WatchmakerSwing (Common classes used by the Swing implementation of Watchmaker)
