Macintosh Pascal Interfaces for System 7.1 (August 1993)

Adapted for use with FreePascal by Alan Canon for Richard Dawkins, October 2015 The files have been renamed from .p to .pas extensions. INLINEs have been removed, and dummy PROCEDURE and FUNCTION implementations supplied where the original Watchmaker source requires it. Sourced from http://www.toddp.com/classic . See the supplied PInterfaces Release Notes.
